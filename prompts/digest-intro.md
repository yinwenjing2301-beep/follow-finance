# 财经日报组装提示词

你在把各个财经人士的摘要拼装成一份完整的日报。

## 格式

用以下标题开头（将 [Date] 替换为今天日期）：

财经日报 — [Date]

然后按以下顺序组织内容：

1. **宏观与市场** — 涉及宏观经济、利率、汇率、大类资产的发言人
2. **股票与个股** — 涉及具体股票、板块、企业的发言人
3. **加密与另类资产** — 涉及比特币、以太坊及其他另类资产的发言人（如有）
4. **其他** — 不好归类的内容

如果某个分区没有内容，直接跳过，不要写空标题。

## 规则

- 只包含有实质内容的发言人，近期无重要发言的跳过
- 每位发言人的摘要直接跟在其名字/身份之后
- 每条内容必须有来源链接，没有链接就不包含
- 不要捏造任何引号、数据或观点
- 最后一行固定添加：

"— 通过 follow-finance skill 生成"

## 链接格式

- 每条摘要后换行附上原始推文链接
- 例如：https://x.com/username/status/xxx
- 如果某发言人有多条推文被引用，每条单独一行列出链接

## 整体风格

- 简洁、信息密集，适合在手机上快速阅读
- 不加任何编辑评论，忠实呈现原始观点
